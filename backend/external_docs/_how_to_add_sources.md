# External RAG Sources

이 폴더는 기업 인사이트 답변의 근거로 사용할 사업보고서, IR 자료, 뉴스 요약 텍스트를 넣는 공간입니다.

권장 파일 예시:
- `samsung_2024_business_report.md`
- `samsung_memory_news_2023.txt`
- `company_ir_2025_q1.md`

권장 내용:
- 회사명
- 기간
- 사업부문별 매출 변화
- 경영진 설명
- 산업 업황
- 환율, 원재료, 판가, 출하량 등 원인 후보

챗봇은 이 폴더의 `.md`, `.txt` 파일을 검색해 재무 추이 분석 답변의 근거 후보로 사용합니다.
