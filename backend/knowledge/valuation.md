# Valuation

- DCF 가치는 미래 자유현금흐름을 할인한 현재가치와 계속가치의 합계이다.
- 기업가치에서 순차입금을 차감하면 자기자본가치가 된다.
- 배수평가는 비교기업의 EV/EBITDA, PER, PBR 등을 활용한다.

## 배당할인모형

- 계속기업을 가정하면 현재 주가는 향후 배당금을 주주의 요구수익률로 할인한 현재가치 합계이다.
- 배당할인모형: `P0 = Σ Dt / (1 + k)^t`
- `Dt`는 t시점 배당금이고, `k`는 주주의 요구수익률 또는 자기자본비용이다.

## 항상성장모형

- 항상성장모형은 기업의 순이익과 배당금이 매년 일정한 비율로 성장한다고 가정한다.
- 배당성향은 `1 - 유보율`이다.
- 성장률은 `g = 유보율 * 재투자수익률(ROE)`이다.
- 1년 말 주당순이익이 `EPS1`일 때 1년 말 주당배당금은 `EPS1 * (1 - b)`이다.
- 항상성장모형 주가: `P0 = EPS1 * (1 - b) / (k - g)`
- 항상성장모형은 주주의 요구수익률 `k`가 성장률 `g`보다 클 때 성립한다.

## 성장기회의 순현재가치

- 성장기회의 순현재가치(NPVGO)는 성장모형 주가에서 무성장모형 주가를 차감한 값이다.
- 무성장모형 주가: `P0 = EPS1 / k`
- `NPVGO = 성장모형 주가 - 무성장모형 주가`
- 주주의 요구수익률이 ROE보다 크면 유보는 주가를 하락시킬 수 있다.
- 주주의 요구수익률이 ROE와 같으면 유보 여부와 무관하게 주가는 변하지 않는다.
- 주주의 요구수익률이 ROE보다 작으면 유보는 주가를 상승시킬 수 있다.

## PER와 PBR

- PER = 현재 주가 / 기대 주당순이익.
- PER = 자기자본 시장가치 / 기대 당기순이익.
- 무성장모형에서 PER = `1 / k`.
- 항상성장모형에서 PER = `(1 - b) / (k - g)`.
- 항상성장모형에서 `PBR = PER * ROE`이므로 PER와 PBR이 주어지면 `ROE = PBR / PER`로 역산할 수 있다.
- 성장률은 `g = ROE * 유보율`이다.
- 항상성장모형 PER 식을 이용하면 `k = g + (1 - b) / PER`로 주주의 요구수익률을 역산할 수 있다.
- PER은 현재 주가를 1년 후 예상 EPS로 나누는 방식과 현재 EPS로 나누는 방식이 있으므로 정의에 주의해야 한다.
- PBR = 현재 주가 / 주당 자기자본 장부가치.
- PBR = 자기자본 시장가치 / 자기자본 장부가치.
- 항상성장모형 예제에서는 `PBR = PER * ROE`로 계산할 수 있다.
- 무성장모형에서 PBR = 1.
- 항상성장모형에서 PBR = `(ROE - g) / (k - g)`.

## 기타 시장가치 비율

- PSR = 현재 주가 / 기대 주당매출액.
- PSR = 자기자본 시장가치 / 기대 매출액.
- PSR = PER * ROS.
- PCR = 현재 주가 / 주당현금흐름.
- PCR = 자기자본 시장가치 / 영업현금흐름.
- Tobin's q = 시장가치 / 대체원가.
- Tobin's q가 1보다 크면 시장에서 구매하는 것보다 직접 만드는 것이 유리하다고 평가할 수 있다.
- EV = 시가총액 + 순차입금.
- 순차입금 = 차입금 - 현금및현금성자산.
- EBITDA = 세전순이익 + 이자비용 + 감가상각비와 무형자산상각비.
- EV/EBITDA = EV / EBITDA.
