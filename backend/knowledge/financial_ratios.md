# Financial Ratios

## 유동성 비율

- 유동성 비율은 기업의 단기채무 지급능력을 측정하는 비율이다.
- 유동비율 = 유동자산 / 유동부채 * 100.
- 유동비율은 단기채무인 유동부채를 현금화가 쉬운 유동자산으로 갚을 수 있는 능력을 나타낸다.
- 유동비율이 지나치게 낮으면 유동성 부족과 부도 위험이 커질 수 있다.
- 유동비율이 지나치게 높으면 자산을 적절히 투자하지 못하고 있을 가능성도 있다.
- 당좌비율 = 당좌자산 / 유동부채 * 100.
- 당좌자산 = 유동자산 - 재고자산 - 선급비용 등.
- 유동비율은 높은데 당좌비율이 낮다면 재고자산을 과다 보유하고 있을 가능성이 있다.
- 현금비율 = 현금및현금성자산 / 유동부채 * 100.
- 현금비율은 가장 보수적인 유동성 비율이다.
- 순운전자본비율 = 순운전자본 / 총자본 * 100.
- 순운전자본 = 유동자산 - 유동부채.

## 안정성 비율

- 안정성 비율 또는 레버리지 비율은 기업이 어느 정도 타인자본에 의존하고 있는지 측정하는 비율이다.
- 부채비율 = 부채 / 자기자본 * 100.
- 자기자본비율 = 자기자본 / 총자본 * 100.
- 이자보상비율 = 영업이익 / 이자비용.
- 이자보상비율은 정상적인 영업활동에서 얻은 영업이익으로 이자비용을 충당할 수 있는지 보여준다.
- 일반적으로 이자보상비율이 1 이하이면 이자 상환능력에 문제가 있는 것으로 볼 수 있다.
- 비유동비율 = 비유동자산 / 자기자본 * 100.
- 비유동비율이 1보다 크면 비유동자산 일부를 부채로 조달한 것으로 해석할 수 있다.
- 비유동장기적합률 = 비유동자산 / (자기자본 + 비유동부채) * 100.
- 비유동장기적합률이 1보다 크면 장기성 자본 외의 추가 자금으로 비유동자산에 투자한 것으로 볼 수 있다.

## 활동성 비율

- 활동성 비율은 기업이 보유한 자산, 부채, 자본을 얼마나 효율적으로 활용하는지 나타내는 지표이다.
- 활동성 비율에서 재무상태표 항목은 일반적으로 연평균 금액을 사용한다.
- 매출채권회전율 = 매출액 / 평균매출채권.
- 매출채권회수기간 = 365일 / 매출채권회전율.
- 매출채권회전율이 높으면 외상매출의 현금화 속도가 빠른 것으로 해석할 수 있다.
- 재고자산회전율 = 매출원가 / 평균재고자산.
- 경우에 따라 재고자산회전율을 매출액 / 평균재고자산으로 계산하기도 한다.
- 재고자산회전기간 = 365일 / 재고자산회전율.
- 재고자산회전기간이 높으면 재고자산이 장기간 판매되지 않는다는 신호일 수 있다.
- 매입채무회전율 = 매입액 / 평균매입채무.
- 경우에 따라 매입채무회전율을 매출원가 / 평균매입채무로 계산하기도 한다.
- 매입채무지급기간 = 365일 / 매입채무회전율.
- 매입채무지급기간이 낮으면 대금 지급이 빠르다는 의미이며 유동성 부담이 커질 수 있다.
- 영업순환주기 = 재고자산회전기간 + 매출채권회수기간.
- 현금순환주기 = 영업순환주기 - 매입채무지급기간.
- 현금순환주기는 외상매입 대금 지급 후 외상매출채권을 현금으로 회수할 때까지 걸리는 기간이다.
- 현금순환주기가 짧을수록 현금 부족 기간이 짧아 유동성이 좋다고 해석할 수 있다.
- 총자산회전율 = 매출액 / 평균총자산.

## 수익성 비율

- 수익성 비율은 기업의 수익을 절대 금액이 아닌 상대적 비율로 나타낸 지표이다.
- 매출총이익률 = 매출총이익 / 매출액 * 100.
- 영업이익률 = 영업이익 / 매출액 * 100.
- 당기순이익률 = 당기순이익 / 매출액 * 100.
- 자기자본이익률 = 당기순이익 / 자기자본 * 100.
- 자기자본이익률은 주주 입장에서 투자한 금액 대비 얻은 투자수익의 개념이다.
- 자기자본이익률은 유보금액에 대한 재투자수익률로도 활용한다.
- 듀퐁분석: `자기자본이익률 = 당기순이익률 * 총자산회전율 * (1 + 부채비율)`.
- 총자산이익률 = 당기순이익 / 총자산 * 100.
- 투하자본수익률 = 영업이익 / 총자본 * 100.

## 성장가능성 계산

- 부채비율을 유지하면 부채는 자기자본에 비례해 증가한다.
- 내부자금만으로 성장하는 경우 차입과 유상증자를 사용하지 않으므로 부채는 변하지 않고, 유보이익만큼 자기자본이 증가한다.
- 유상증자를 사용하지 않고 부채비율을 유지하는 경우 자기자본은 유보이익만큼 증가하고, 부채는 목표 부채비율에 맞춰 조달할 수 있다.
- 최대 성장률을 구할 때 유보율 정책을 바꾸는 것이 허용되면 유보율 100%를 적용해 내부성장 가능 범위를 계산한다.

## 분석 기준

- 비율 분석은 동종 업계, 과거 추세, 회사 목표와 비교해야 한다.
