# Time Value of Money

## 기본 개념

- 같은 금액이라면 미래의 현금보다 현재의 현금을 선호하는 현상을 유동성 선호라고 한다.
- 유동성 선호의 이유는 시차 선호, 투자 기회, 물가 상승, 불확실성으로 정리할 수 있다.
- 현재 현금을 포기하는 경우 그에 상응하는 대가를 요구하게 되며, 이를 비율로 표현한 것이 이자율이다.
- 재무관리에서는 일반적으로 원금뿐 아니라 이자에도 매 기간 이자가 붙는 복리 기준으로 가치를 산정한다.

## 현재가치와 미래가치

- 서로 다른 시점의 현금흐름은 동일한 시점의 가치로 환산해야 비교할 수 있다.
- 재무관리에서는 현재 시점의 의사결정이 많으므로 미래 현금흐름을 현재가치로 환산해 비교한다.
- 미래 금액을 현재가치로 환산하는 것을 할인이라고 하며, 이때 적용하는 이자율을 할인율이라고 한다.

## 기본 공식

- 미래가치: `FV = PV * (1 + r)^n`
- 현재가치: `PV = FV / (1 + r)^n`
- 미래가치 이자요소: `FVIF(r, n) = (1 + r)^n`
- 현가 이자요소: `PVIF(r, n) = 1 / (1 + r)^n`

## 일반연금

- 연금은 일정한 금액이 일정 기간 동안 반복적으로 발생하는 현금흐름이다.
- 일반연금은 매 기간 말에 현금흐름이 발생하는 연금이다.
- 일반연금 현재가치: `PV = PMT * [1 - 1 / (1 + r)^n] / r`
- 일반연금 미래가치: `FV = PMT * [(1 + r)^n - 1] / r`

## 영구연금과 성장영구연금

- 무성장 영구연금 현재가치: `PV = PMT / r`
- 일정성장 영구연금 현재가치: `PV = C1 / (r - g)`
- 일정성장 영구연금 공식은 할인율 `r`이 성장률 `g`보다 클 때 성립한다.
- 매년이 아니라 매 2년마다 현금흐름이 발생하는 영구연금은 2년 기간 이자율을 사용한다.
- 예를 들어 연 APR 10%의 연간 이산복리에서 매 2년마다 1,000원이 유입되면 2년 기간 이자율은 `(1 + 10%)^2 - 1 = 21%`이고 현재가치는 `1,000 / 21%`이다.
- 일정 기간 높은 성장률로 성장한 후 낮은 성장률로 영구 성장하는 경우, 명시 예측기간 현금흐름을 각각 할인하고 마지막 명시기간 말의 terminal value를 더해 할인한다.
- 단계성장 영구연금 terminal value: `TV_t = C_(t+1) / (r - g)`

## 기초연금과 기말연금

- 기말연금은 매 기간 말에 지급받는 일반연금이다.
- 기초연금은 매 기간 초에 지급받는 선불연금이다.
- 동일한 현금흐름과 기간이라면 기초연금의 가치는 기말연금의 가치보다 `(1 + r)`배 크다.
- 기초연금 가치: `기말연금 가치 * (1 + r)`

## 표시이자율과 실효이자율

- 연 표시이자율(APR)은 시장에 표시되는 연 단위 이자율이다.
- 1년보다 짧은 기간마다 이자가 발생하면 기간당 이자율은 `APR / m`이다.
- 연 실효이자율(EAR)은 복리 효과를 반영한 1년간의 실제 수익률이다.
- APR과 EAR의 관계: `1 + EAR = (1 + APR / m)^m`
- 연속복리에서는 복리 횟수를 무한대로 가정한다.
- 연속복리 1년 기준: `1 + EAR = e^APR`

## 표시이자율과 실효이자율을 이용한 현재가치

- 현금흐름이 1년보다 짧은 주기로 발생하면 표시이자율을 기간당 이자율로 나누어 할인할 수 있다.
- 같은 현금흐름을 실효이자율로 할인해도 일관된 기간 변환을 적용하면 같은 현재가치가 나온다.

## 채권과 연속복리

- 이표채의 현재가치는 각 이자지급액의 현재가치와 만기 액면가의 현재가치 합계이다.
- 6개월마다 이자를 지급하는 채권은 연 표시 시장이자율을 2로 나눈 6개월 기간 이자율로 할인한다.
- 연속복리에서는 `t`년 후 현금흐름의 현재가치를 `CF / e^(r*t)`로 계산한다.
- 이자 지급 횟수가 무한대인 영구채는 연간 이자액을 연속복리 할인율로 나눈 값과 동일하게 평가할 수 있다.
